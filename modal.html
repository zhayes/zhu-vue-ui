# Modal

全局注册

    import { Modal } from 'zhu-vue-ui';
    Vue.use(Modal);
1
2

全局注册后，可在组件内部调用$modal方法。该方法返回一个带destory方法的对象，可供销毁组件实例。

    <script>
    export default {
        methods:{
            showModal(){
                const m = this.$modal({
                    title:"温馨提示"
                    content:"hello,world!",
                    onOk(){
                        //...
                    },
                    onCancel(){
                        m.destroy();
                    }
                })
            }
        }
    }
    </script>

# options

optoin说明类型默认
title标题string提示
content提示内容string无
cancelText取消按钮文案string取消
okText确认按钮文案string确定
okTextColor确认按钮文案颜色string#3a8cf4
cancelTextColor取消按钮文案颜色string999999
maskDestroy点击mask蒙板是否关闭booleantrue
onOk定义边框线样式Function无
onCancel定义边框线样式Function无
